

🎓 Student Marks Management System (JSP CRUD Project)
📌 Project Overview

The Student Marks Management System is a Java-based web application developed using JSP, JDBC, and PostgreSQL.
This project enables administrators or faculty to Create, Read, Update, and Delete (CRUD) student academic records through a secure and user-friendly web interface.

The application demonstrates core Java web development concepts, database connectivity, and MVC-style separation using JSP pages.

🚀 Key Features

Student Login Authentication

> Add Student Marks

View Student Marks

Update Student Information

Delete Student Records

Responsive UI using Bootstrap

Secure database operations using JDBC
